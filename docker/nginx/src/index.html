LEAF Development Environment

This development environment contains copies of the Request Portal and Nexus components.

General Development

This can be used for general development and testing.

Request Portal A low code/no code workflow management tool utilized to digitize administrative business processes
Nexus Manage facility-level user groups and organizational charts.

Standardized Testing

This is primarily used for automated testing, where code changes can be checked against the automated test suite.

Prerequisite: Populate the database by running the API tester.

⚠️ Warning: The API tester will reset the contents of these sites.

  • Request Portal
  • Agent Task Repository
    • Repository of tasks for the LEAF Agent
  • Nexus

Automated Tests

Deployment requires all tests to be passing

API tester These are tests primarily for api's that can be targeted by client-side javascript calls.
Run in verbose mode
End-to-End tests (Playwright) New end-to-end test suite. Prerequisite: Run the API tester in order to initialize the test database.
LLM tester Test Large Language Model responses for expected behavior.
Run in verbose mode

Dev Corner
